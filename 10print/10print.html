Ocean 10Print

Overview

Ocean 10Print is a generative text piece that fills the screen with a continuously growing field of symbols, producing an endlessly scrolling visual texture that evokes the surface of water. Built in JavaScript as a remix of the classic 10Print algorithm (originally written in BASIC for the Commodore 64) ,the piece replaces the algorithm's characteristic diagonal slashes with text-based glyphs sourced from emoji searches around the word "ocean," assembling them into a sparse, living composition.

Concept & Inspiration

The 10Print algorithm is one of the most well-known examples of emergent complexity from simple rules: a single line of code that produces infinitely varied, maze-like patterns through randomness. This piece takes that logic and redirects it toward a specific aesthetic and emotional landscape. The classic 10Print color palette is retained as the base, a flat

t#0082c8 blue that grounds the piece, while occasional glyphs break into darker, more varied ocean tones. The result is a field of texture and language that dissolve into a pattern of the sea.

Process

The character set was assembled by searching through ocean-related text emoji and selecting glyphs that carried the right visual weight: ⋆, ˚, ., ᐟ, 𓂃, ﹏, ~, `, ⣠, ଳ, 𓂁, 𓏲𓂃, 𓈒𓏸.°•. Each frame, one glyph is selected at random from this set using nn.random().

Color is generated by randomizing the red and green channels within a narrow low range while keeping blue dominant, producing subtle variation within the oceanic palette.

A 12% probability threshold determines whether a glyph receives this varied color or falls back to the base blue - so most of the field stays anchored while occasional glyphs surface in darker tones, mimicking the way light catches water unevenly.

The piece scrolls continuously downward as new glyphs are appended, never repeating and never stopping.
